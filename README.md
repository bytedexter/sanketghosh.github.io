# Sanket Ghosh — Portfolio

Personal portfolio website built with clean, minimal HTML, CSS, and JavaScript.

## Sections

- **Hero** — Name, tagline, and quick intro
- **About** — Bio and key stats
- **Experience** — Work history timeline
- **Projects** — Featured project cards
- **Publications** — Research papers
- **Skills** — Technical skill tags
- **Education** — Academic background
- **Contact** — Email, phone, LinkedIn, GitHub

## Development

Open `index.html` in a browser, or use a local server:

```bash
python -m http.server 8000
```

## Deployment

Deploy to GitHub Pages, Vercel, or Netlify by connecting this repository.
